# Cafe24 호스팅 배포 가이드

이 폴더(`cafe24_build`)의 모든 파일을 Cafe24 웹 호스팅 서버의 `www` 폴더(또는 `public_html`) 안에 업로드하세요.

## 1. 파일질라(FileZilla) 등 FTP 프로그램 접속
- 호스트: (Cafe24에서 제공받은 FTP 주소)
- 사용자명: (Cafe24 아이디)
- 비밀번호: (FTP 비밀번호)

## 2. 파일 업로드
- 서버의 `www` (또는 `public_html`) 폴더 안에 **`lotto` 폴더**가 있다면 그 안으로 들어가세요. (없다면 만드세요)
- `cafe24_build` 폴더 **안에 있는** 모든 파일과 폴더를 `lotto` 폴더 안으로 업로드하세요.
- **주의**: `cafe24_build` 폴더 자체를 올리는 것이 아니라, 그 **내용물**을 `lotto` 폴더 안에 넣어야 합니다.

## 3. 권한 설정 (필요시)
- `data` 폴더와 그 안의 `lotto.db` 파일에 쓰기 권한(777 또는 707)이 필요할 수 있습니다.
- FTP 프로그램에서 `data` 폴더 우클릭 -> 파일 권한 -> 숫자 값에 `777` 입력 -> 확인.

## 4. 접속 확인
- 브라우저에서 `http://(아이디).cafe24.com` 으로 접속하여 확인합니다.

## 주의사항
- 이 버전은 PHP 환경에서 동작하도록 재작성된 버전입니다.
- 데이터베이스는 파일 기반인 SQLite(`data/lotto.db`)를 사용하므로 별도의 DB 설정이 필요 없습니다.
